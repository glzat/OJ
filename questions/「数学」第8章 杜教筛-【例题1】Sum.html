题目描述

题目背景

小 C 和小 L 在玩 Jenga 游戏。

Jenga 游戏的中文名叫层层叠（叠叠乐、抽积木、积木游戏等），是一个考验动手能力和大脑思维的游戏，在游戏中，玩家交替从积木塔中抽出一块积木并且使其平衡的放到塔顶，去创造一个不段增高，越来越失去根基的积木塔，直到积木塔倾倒。

题目描述

注意，本题中的 Jenga 游戏的规则与传统的 Jenga 游戏的规则 相类似，但有所区别，比如，本题的游戏去除了将抽出的积木放到塔顶的过程 。

本题的游戏在一座 层的积木塔上进行，每一层都有三根长方体的积木，同层的积木相互平行，相邻两层的积木相互垂直。

两人轮流玩游戏。轮到某个人时，这个人必须在这一轮 移走 一根积木，并且保证 移走这一根积木后积木塔平衡 （可以从 任意一个至少两根积木的层 移走积木）。无法操作的玩家输掉游戏。

积木塔能保持平衡当且仅当下面三个条件同时满足：

  • 每一层至少有一根积木。
  • 如果有一层只有一根积木，那必须是中间的那一根。
  • 不存在相邻两层都只有一根积木（注意这个条件和传统的 Jenga 游戏不同） 。

小 C 和小 L 玩了很多次这个游戏，每一次都是小 C 先手，但是小 C 因为水平不够，而小 L 始终采取最优策略，小 C 每一次游戏都输给了小 L。

但是小 C 不甘心，他想要还原这几次游戏，他想知道，假设他能通过努力提高自己的水平，自己能否战胜小 L。

于是现在小 C 给出了每次游戏的初始状态，每个初始状态都是由完整的积木塔移走一些积木形成的，保证给出的初始状态是平衡的。对于每次游戏，小 C 都想知道，在两人都采取最优策略时，谁能获胜。

显然小 C 不懂得回答这些问题，请你帮他解决这些不难的问题。

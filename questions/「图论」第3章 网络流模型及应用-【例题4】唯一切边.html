题目描述

有一个 的棋盘，可以在上面放棋子。

有些格子不能放棋子，有些格子必须放棋子，剩下的格子随意。

要求放好棋子之后满足如下两条要求：

  1. 第 行和第 列的棋子数目必须一样多。
  2. 第 行的棋子数目不能超过总的棋子数目的 。

求最多可以另外放多少个棋子（就是除掉必须放的）。如果无解输出 impossible 。

输入格式

对于每组数据，第一行包括三个正整数 。

接下来包含 行,每行包含 个字符，表示了描述棋盘的矩阵。其中 . 表示可用格子， / 表示不可用格子， C 表示格子已被一个棋子占据。

整个测试以 0 0 0 表示结束。

输出格式

如果有解，输出一行包括一个正整数，表示最多能再安装多少个组件。否则，输出 impossible 。具体格式如样例所示。

2 1 1
/.
//

2 100 100

C.

CC/..
././/
..C.C
/.C..
/./C/
5 2 10
CC/..
././/
..C.C
/.C..
/./C/
0 0 0

样例输出 

Case 1: 0
Case 2: 1
Case 3: impossible
Case 4: 7
Case 5: impossible

题目描述

某天，小明按照地图来到了一片洞穴群地区，这片区域由 个洞穴（分别编号为 到 ）以及若干通道组成，并且每条通道连接了恰好两个洞穴。假如两个洞穴可以通过一条或者多条通道按一定顺序连接起来，那么这两个洞穴就是连通的，按顺序连接在一起的这些通道则被称之为这两个洞穴之间的一条路径。通道不太稳定，时常因为外界影响而发生改变，比如 号洞穴和 号洞穴之间有时会出现一条通道，有时这条通道又会因为某种稀奇古怪的原因被毁。

小明发现：无论通道怎么改变，任意时刻任意两个洞穴之间至多只有一条路径。

小明有一台监测仪器可以实时将通道的每一次改变状况在小明手边的终端机上显示：

如果监测到洞穴 和洞穴 之间出现了一条通道，终端机上会显示一条指令 Connect u v ；

如果监测到洞穴 和洞穴 之间的通道被毁，终端机上会显示一条指令 Destroy u v ；

小明希望能随时通过终端机发出指令 Query u v ，向监测仪询问此时洞穴 和洞穴 是否连通。现在你要为他编写程序回答每一次询问。已知在第一条指令显示之前，洞穴群中没有任何通道存在。

输入格式

第一行为两个正整数 和 ，分别表示洞穴的个数和终端机上出现过的指令的个数。

以下 行，依次表示终端机上出现的各条指令。每行开头是一个表示指令种类的字符串 （ Connect 、 Destroy 或者 Query ，区分大小写），之后有两个整数 和 分别表示两个洞穴的编号。

输出格式

对每个 Query 指令，输出洞穴 和洞穴 是否互相连通。如果连通，输出 Yes ，否则输出 No 。

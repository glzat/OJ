题目描述

带插入、修改的区间 小值在线修改查询。

要求支持以下三种操作：

  1. Q x y k : 询问 区间内的第 小数值

  2. M x val : 把位置在 上的数的数值修改为 。

  3. I x val : 把位置在 上的数前面加入一个数值为 的数。即现在所在 位置上的数就是刚插入的数。

输入格式

第一行一个正整数 ，表示原来有 个数。

第二行有 个用空格隔开的非负整数。

第三行一个正整数 ，表示下面有多少个操作。

下面一共 行，一共三种对原序列的操作：（假设此时一共 个数）

  1. Q x y k : 询问 区间内的第 小数值

  2. M x val : 把位置在 上的数的数值修改为 。

  3. I x val : 把位置在 上的数前面加入一个数值为 的数。即现在在位置 上的数就是刚插入的数。

为了体现在线操作，设 为上一次查询的时候程序输出的结果，如果之前没有查询过，则 lastAns = 0 。 则输入的时候实际是：

Q x y k 要先将 x^=lastAns y^=lastAns k^=lastAns

M x val 要先将 x^=lastAns val^=lastAns

I x val 要先将 x^=lastAns val^=lastAns

简单来说就是操作中输入的整数都要异或上一次询问的结果进行解码。

输出格式

对于每个询问输出回答，每行一个回答。

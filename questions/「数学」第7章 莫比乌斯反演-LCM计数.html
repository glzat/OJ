题目描述

给定正整数 ，请你求出如下式子的值：

答案对 取模。

输入格式

一行一个正整数 。

输出格式

一行一个整数表示上述式子对 取模的值。

样例

样例输入1 

987654

样例输出1 

3595606

样例输入2 

98723489637846786423

样例输出2 

765547726

数据范围与提示

本题采用捆绑测试。

子任务编号 分值

考虑到 的范围超过 long long ，因此变量类型需要采用 __int128 （其范围是 ）。而 C++ 内并没有兼容 __int128 的输入方式。下面给出用于输入 __int128 的函数： 

template <class Tp>
void read(Tp &x) {
	static char ch;
	static bool neg;
	for (ch = neg = 0; ch < '0' || ch > '9'; neg |= (ch == '-'), ch = getchar());
	for (x = 0; ch >= '0' && ch <= '9'; (x *= 10) += (ch ^ 48), ch = getchar());
	neg && (x = -x);
}

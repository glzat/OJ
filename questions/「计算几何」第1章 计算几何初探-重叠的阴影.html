题目描述

信用卡是一个矩形，但矩形的四个角做了圆滑处理，使它们都是与矩形的两边相切的 圆，如下图所示。

现在平面上有一些规格相同的信用卡，试求其凸包的周长。注意凸包未必是多边形，因为它可能包含若干段圆弧。

输入格式

输入的第一行是一个正整数 ，表示信用卡的张数。

第二行包含三个实数 ，分别表示信用卡（圆滑处理前）竖直方向的长度、水平方向的长度，以及 圆的半径。

之后 行，每行包含三个实数 ，分别表示一张信用卡中心（即对角线交点）的横、纵坐标，以及绕中心逆时针旋转的弧度。

输出格式

输出只有一行，包含一个实数，表示凸包的周长，四舍五入精确到小数点后 位。

pascal 

uses math;
const Pi = 3.141592653589793;
begin
writeln(sin(30.0 / 180.0 * Pi) : 0 : 10);
writeln(cos(60.0 / 180.0 * Pi) : 0 : 10);

end.

C++ 

#include <iostream>
#include <math.h>

const double Pi = 3.141592653589793;

{

    cout << asin(1.0) << endl;
    cout << acos(0.0) << endl;
    cout << atan(1.0) << endl;
    return 0;
}
